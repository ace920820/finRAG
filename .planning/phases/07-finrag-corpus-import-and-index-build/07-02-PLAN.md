<threat_model>
## Threat Model

### Assets
- End-to-end local ingestion command.
- Retrieval index consistency with processed chunks.
- Existing API/query contract.

### Threats
- **High:** Index rebuild loads stale chunks or stale vectors.
  - Mitigation: make import/index commands accept explicit processed/index dirs or clear stale index files before rebuild.
- **High:** Integration tests mutate committed fixture/index data.
  - Mitigation: use temporary directories/settings/env and restore caches.
- **Medium:** CLI path handling breaks external-volume or Chinese paths.
  - Mitigation: use `Path`, not shell splitting; add tests with non-ASCII names if practical.
- **Medium:** API test reads old cached settings/loaders.
  - Mitigation: clear relevant lru caches in tests.

### Blocking Requirements
- Do not require live Alibaba Bailian credentials.
- Do not change response models for `GET /api/documents` or `POST /api/query`.
- Do not remove demo fixture fallback.
</threat_model>
