<threat_model>
## Threat Model

- **[high] Table numeric evidence corruption** — Preserve raw table fact fields and avoid lossy summarization for numeric facts.
- **[high] Citation drift** — Keep citation IDs stable for retained evidence and build citations from the same evidence used in generation.
- **[medium] Prompt under-contextualization** — Compact text conservatively and keep source/title/company/date/page metadata.
- **[medium] SSE contract drift** — Preserve event names/order and keep any new metadata additive.
- **[medium] Runtime/provider creep** — Evidence compression must be deterministic and not require extra model/API calls.
- **[medium] Later-phase scope creep** — Do not implement iterative retrieval or hierarchical drill-down.
</threat_model>
