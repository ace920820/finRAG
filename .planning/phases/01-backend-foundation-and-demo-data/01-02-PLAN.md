<threat_model>
## Threat Model

### Assets
- Demo financial facts and citation-ready source chunks.
- Local processed fixture data consumed by backend API.

### Threats
- **Medium:** Fixture data may contain untraceable financial claims that later cause citation hallucinations.
  - Mitigation: every critical fact must belong to a chunk with doc/chunk metadata and source title.
- **Low:** Path traversal or arbitrary file reads from configurable data paths.
  - Mitigation: loaders should read known JSON filenames from configured processed data directory only.
- **Low:** Generated data scripts overwrite unexpected files.
  - Mitigation: scripts write only under `backend/app/data/processed` unless explicitly configured.

### Blocking Requirement
No script may fetch remote documents or require network access in Phase 1.
</threat_model>
