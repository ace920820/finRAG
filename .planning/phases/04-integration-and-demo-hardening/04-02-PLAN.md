<threat_model>
## Threat Model

### Assets
- Imported frontend UI/interaction design.
- Demo query flow.
- Citation and retrieval visibility.

### Threats
- **High:** Rewriting components breaks externally delivered frontend design.
  - Mitigation: only change props/state wiring; preserve existing markup/classes as much as possible.
- **High:** Concurrent query streams corrupt active assistant message.
  - Mitigation: disable or abort prior stream when a new query starts; update by assistant message ID.
- **Medium:** Backend failures leave UI stuck in loading stage.
  - Mitigation: handle thrown fetch errors and SSE `error` events by marking assistant message done with error content.
- **Medium:** Document library is empty if backend is unavailable.
  - Mitigation: fallback to existing mock document list for display only.

### Blocking Requirement
Do not implement frontend visual redesign, new routing, or broad state-management refactors.
</threat_model>
