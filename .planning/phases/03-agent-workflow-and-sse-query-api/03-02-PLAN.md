<threat_model>
## Threat Model

### Assets
- SSE event contract consumed by frontend.
- Backend stack traces and internal errors.
- Demo reliability during streamed response.

### Threats
- **High:** Malformed SSE frames break frontend integration.
  - Mitigation: centralize formatting in `backend/app/core/sse.py` and test event parsing.
- **High:** Exceptions expose stack traces or hang the stream.
  - Mitigation: catch route-level exceptions and emit `error` event with code/message.
- **Medium:** Event order changes make frontend adapter brittle.
  - Mitigation: endpoint tests assert required event order.
- **Medium:** Streaming chunks split citation tags incorrectly.
  - Mitigation: chunk at conservative boundaries or small paragraph boundaries where possible.

### Blocking Requirement
The endpoint must be testable through FastAPI `TestClient` without network calls or API keys.
</threat_model>
