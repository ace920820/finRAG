<threat_model>
## Threat Model

### Assets
- Retrieval/rerank debug outputs.
- Chunk content and citation metadata.
- Optional Bailian API key if live rerank is manually enabled.

### Threats
- **Medium:** Debug endpoint becomes treated as production contract.
  - Mitigation: document as dev-only and keep formal frontend flow assigned to Phase 3 SSE.
- **Medium:** Rerank provider exceptions leak stack traces or secrets.
  - Mitigation: catch provider failures, fallback to fused Top 5, and return safe degradation metadata.
- **Low:** Debug endpoint exposes too much raw data.
  - Mitigation: return only fixture/demo chunk content and metadata required for retrieval visualization.

### Blocking Requirement
Debug endpoint and tests must run without live provider credentials.
</threat_model>
