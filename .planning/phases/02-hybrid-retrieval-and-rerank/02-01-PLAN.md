<threat_model>
## Threat Model

### Assets
- Alibaba Cloud Bailian API credentials.
- Model provider configuration.
- Deterministic test behavior.

### Threats
- **High:** API keys accidentally committed in `.env` or tests.
  - Mitigation: commit only `.env.example` with blank values; never write real secrets.
- **Medium:** Tests unexpectedly call live APIs.
  - Mitigation: mock provider is default for tests and no live call runs without explicit config/mode.
- **Medium:** Provider-specific SDK details leak into retrieval services.
  - Mitigation: define provider protocols/classes and inject provider instances into retrieval/rerank services.

### Blocking Requirement
No real API key or secret may be committed, logged, or required for tests.
</threat_model>
